Visual API Builder

A full-stack low-code platform to design, document and export REST APIs through a drag-and-drop visual canvas — no backend code required. Inspired by n8n, Hasura and Retool. Generates a complete NestJS project, Prisma schema, OpenAPI spec, Postman collection and Docker setup from a metadata graph.

KEY FEATURES

  • Drag-and-drop node editor powered by React Flow — Request, Auth, Validation, DB Query, Response nodes
  • Entity designer with typed fields, constraints and relations
  • Code generation: NestJS, Prisma, OpenAPI 3.0, Postman collection, Docker
  • JWT auth with role-based access (admin / developer / viewer) and refresh tokens
  • In-app API playground — test endpoints without leaving the browser

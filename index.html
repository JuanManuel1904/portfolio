CryptoDash — Real-time Crypto Dashboard

Full-stack dashboard that streams live cryptocurrency prices via WebSockets. Built with a FastAPI backend that polls CoinGecko every 10 seconds and broadcasts to all connected clients simultaneously. Includes coin search across the entire CoinGecko catalog, interactive price charts, and dark-first UI.

KEY FEATURES

  • Real-time WebSocket broadcast — prices update every 10 seconds across all clients
  • Search any crypto by name or symbol across the full CoinGecko catalog (+10,000 coins)
  • Interactive line chart with 60-point price history and custom tooltip
  • Price flash animation (green/red) on every tick update
  • Dockerized FastAPI backend with auto-reconnecting WebSocket on the client
